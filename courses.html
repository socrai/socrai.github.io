Program

Our program provides a comprehensive way for students to learn all about machine learning and do productive research! Read more to see how we accomplish this!

Apply
Mathematical Fundamentals

Students will grasp foundational concepts in linear algebra, calculus, probability, and statistics essential for understanding machine learning algorithms.

Model Training and Finetuning

Learners will understand how to train and optimize machine learning models using techniques like hyperparameter tuning, transfer learning, and regularization.

Python Libraries for Model Training

Exploration of key Python libraries including TensorFlow, PyTorch, Keras, scikit-learn, XGBoost, and LightGBM for effective model development and optimization.

Working with Big Data

Students will learn to handle large datasets, tackling challenges like volume and variety, to extract meaningful insights and patterns.

Python Libraries for Big Data

Proficiency in Python libraries such as Pandas, Dask, PySpark, Hadoop Streaming, Vaex, and HDF5 for efficient processing and analysis of big data.
